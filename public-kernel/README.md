# Fractal Decision Ecosystem Public Kernel

Fractal Decision Ecosystem（FDE） is a reduced public reference kernel for
routing AI-assisted work through evidence, scope, public-safety, and completion
checks.

This public kernel is intentionally small. It does not disclose the private FDE
operating system, full recursive skill layer, source-pointer registry, private
workflows, absorbed dialogues, generator internals, or patent-candidate
implementation details.

## Public Kernel

FDE uses a three-layer operating shape:

1. Core: protect facts, scope, publication boundaries, and completion claims.
2. Router: choose the correct lane, runtime, or public/private path.
3. Leaf: point to the minimum source needed for the task.

The public kernel exposes four general gates:

- Pre-execution fact check
- Scope routing
- Publication containment
- Done verification closeout
