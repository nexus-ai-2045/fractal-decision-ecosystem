# Edge Cases

- Linear is requested but no Linear tool, CLI, or token is available: use optional manual handoff only and do not create a GitHub issue as a substitute.
- Chrome can reach external services: final create / submit remains an external write action and needs explicit approval.
- Public release is requested after local MVP is green: stop at publication-containment gate and restate target repository, exact operation, checks, and web visibility risk.
- Patent filing is requested: treat filing as a separate external legal action; local packet readiness is not filing completion.
- User asks for `残務ゼロ`: answer implementation residue, operation residue, optional external residue, and publication approval separately.
